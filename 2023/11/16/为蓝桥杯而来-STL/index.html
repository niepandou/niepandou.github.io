为蓝桥杯而来-STL

发表于2023-11-16|更新于2023-11-16
|字数总计:2.2k|阅读时长:9分钟|阅读量:

STL二次学习

列表

系统为某一程序分配空间时，所需时间与空间大小无关，与申请次数有关

vector

代码示范

vector支持比较运算，按字典序比较

pair

存储二元组

也能够支持比较运算，按字典序比较，以first为第一关键字，second为第二关键字

string

substr的第二个参数代表长度

queue

queue没有clear函数

priority_queue

优先队列，即堆,默认大根堆

改成小根堆的两种方法

变号法

定义法

stack

栈

deque

双端队列，加强版vector

set/multiset

set定义数组不允许出现重复元素，如果出现就会忽略掉

multiset是可以有重复元素的

所有set操作的时间复杂度都为logN

迭代器里两个名词的意思

map/multimap

map存储的是一种映射关系

同样可以用insert和erase

insert可以用pair插入

erase的参数可以是pair也可以是迭代器

find操作也和set一样

set操作能做的map大多都能做

实用操作，可以让任意类型作为下标，(时间复杂度为O（logN）)

unordered

优化set/multiset和map/multimap的时间复杂度，基于哈希表，使大多数操作的时间复杂度都为O（1）

但不支持lower_bound和upper_bound

，也不支持迭代器相关的操作

bitset

可以比正常的内存占用省掉八倍

c++中布尔数组为1字节，1024个字节也就是1kb内存

如果用bitset压位的话，让布尔数组每一个位都能被充分利用，那么就只需128个字节

bitset也支持位运算操作，也支持count操作
